Frequently Asked Questions

Accounts and Login Issues

Q: How to reset the password?

A: Go to the settings page and click "Change Password".

Q: What if I forgot my login ID?

A: Contact support to retrieve your login ID.

Account Management

Q: How to add new admins?

A: Go to the settings page and use the "Add Admin" form.

Q: How to create new accounts?

A: Go to the Create Users in dashboard nav-bar and create accounts for faculty/student/parent.

Q: How to change password of an user in case if he/she forgets?

A: Admin can't change the password of Leviosa app users directly, instead delete the account and create new account.

Troubleshooting

Issue: The application is running slowly.

Solution: Try clearing your browser cache or restarting the app.

Issue: The app is crashing when I try to open it.

Solution: Ensure that your device has the latest version of the app.

Issue: I can't upload any files or assignments.

Solution: Check your internet connection and ensure that the file size is within the allowed limits.

Issue: I am unable to create or edit assignments as a teacher.

Solution: Make sure you are logged in with the correct admin or teacher account. If the problem continues, contact the system administrator or support for further assistance.

Issue: Some sections of the app are not loading.

Solution: This could be due to a temporary issue with the app’s servers. Try reloading the page or app. If the problem persists, contact support for help.
